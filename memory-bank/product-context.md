# Product Context

This project is a sophisticated web application for managing AI bots, campaigns, and chat sessions. It provides a dashboard for tracking engagement and tools for configuring bot behavior.

## Core Purpose
- Manage and deploy AI bots.
- Create and track marketing campaigns.
- Monitor and interact with chat sessions in real-time.
- Manage users and system settings.

## User Goals
- Efficiently configure AI agents.
- View analytics on campaign performance.
- Seamlessly manage multiple chat interactions.
- Scalable and maintainable system for growth.
